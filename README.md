## InReach

[NestJS](https://github.com/nestjs/nest) app for location check-ins. 

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Google Auth

Visit [http://localhost:3000/auth/google](http://localhost:3000/auth/google) to init Google Auth
